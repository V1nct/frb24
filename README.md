## 使用方法

1. 使用 Xelatex 编译 `main.tex`, 路径不要有多字节字符.
2. 目录和引用可能要编译两次才会出现.
3. 分节内容可以放在 `./subtexs` 中
4. 图片放在 `./pictures`, 由于总文件大小限制, 图片一定要压缩
5. 编译例子和教程放在 `./examples`
